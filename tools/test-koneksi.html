🔌 Tes Koneksi Fase 0

Halaman ini membuktikan asumsi teknis paling berisiko sebelum Anda membangun/men-deploy seluruh situs: apakah GitHub Pages dapat membaca respons GAS (CORS), mengirim POST, dan menampilkan gambar Google Drive.

Cara pakai paling sahih: unggah file ini ke GitHub Pages Anda, lalu buka dari URL *.github.io (bukan dari komputer) agar kondisi CORS-nya nyata.

Interpretasi: Jika ketiganya ✅ hijau, arsitektur aman — lanjut build/deploy normal. Jika GET gagal tapi POST ok, set USE_JSONP_FOR_GET = true di js/config.js (fallback JSONP). Jika gambar gagal walau sudah di-share + referrerpolicy, pertimbangkan host gambar lain.
